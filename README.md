# PyPhone Frontend

The React frontend for PyPhone OS — a web-based mobile OS simulator.

## Known Issues

### Nexus AI
- **Image Generation**: Currently unavailable. Image generation requests will not produce results.
- **Web Search**: Currently experiencing issues. Search-grounded responses may not work correctly.

### Dev Studio
- **JavaScript**: Code execution in the output panel is currently not working properly.
- **Python**: The Python runner does not support user input (`input()` is not supported).
- **C++**: C++ code execution is currently not working.
